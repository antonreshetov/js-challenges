# JavaScript challenge

Learn JS by solving coding exercises.

## Get Started

```sh
# Install dependencies
npm install
# or
pnpm install

# Start challenge <category>/<number>
npm run c basics/01

# Start solution <category>/<number>
npm run c basics/01 -- -s
```
